## 個人情報の利用目的

- 収集した情報は、不具合の修正、アクセス解析、サービス改善のために利用します。

- 当サイトでは、Google Analytics 及び Amazon CloudWatch RUM を使用しています。これにより収集されるデータは匿名であり、個人を特定するものではありません。
